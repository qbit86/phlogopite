# Changelog

## [0.2.0] - 2019-07-29
### Removed
- `IFormattedSink<TProperty>`
- `FormattingSink`

## [0.1.0] - 2019-06-01
### Added
- `IFormattedSink<TProperty>` as abstraction for text-based sinks.
- `FormattingSink` as default implementation of proxy between mediator and text-based sinks.
